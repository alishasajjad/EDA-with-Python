# 📊 EDA with Python

Welcome to the **EDA with Python** repository.

This repository is dedicated to learning and practicing **Exploratory Data Analysis (EDA)** using Python. It contains practical notebooks, sample datasets, and real-world examples that demonstrate how to inspect, clean, analyze, and visualize data.

The goal of this repository is to build a strong foundation in data analysis, improve data exploration skills, and prepare for more advanced topics in data science and machine learning.
